# 🎡 Карусель занятий (Space Edition) 🚀

Иммерсивное веб-приложение "Колесо Фортуны", созданное специально для детей. Это не просто список дел, а настоящий мостик космического корабля, который помогает выбрать следующее увлекательное занятие.

![App Preview](https://raw.githubusercontent.com/zemuro/fun-carousel/main/favicon.png)

## 🌌 Особенности проекта

*   **3D Star Tunnel**: Динамический фон с эффектом гиперпрыжка, созданный на HTML5 Canvas.
*   **Эффект иллюминатора**: Интерфейс "кабины пилота" с глубокими тенями и эффектом защитного стекла.
*   **Neon Cyber Style**: Полупрозрачные "стеклянные" элементы интерфейса с неоновым свечением.
*   **Интерактивный звук**: Полная озвучка действий (клики, щелчки колеса, победные фанфары) через Web Audio API.
*   **Умная верстка**: Автоматическая подстройка размера шрифта на колесе (поддерживает до 30+ элементов без потери читаемости).
*   **Полная автономность**: Работает без интернета, не использует внешние библиотеки (кроме конфетти).

## 🛠 Технологии

*   **Frontend**: HTML5, Vanilla CSS (Glassmorphism), JavaScript.
*   **Graphics**: Canvas API (Starfield & Wheel drawing).
*   **Audio**: Web Audio API (синтез звука на лету).
*   **Data**: LocalStorage для сохранения категорий и игр.

## 🚀 Как запустить

### Локально (Windows)
1.  Скачайте папку с проектом.
2.  Запустите `start-server.bat`. Приложение откроется автоматически по адресу `http://localhost:8088`.
3.  Нажмите на иконку шестеренки ⚙️ в углу, чтобы настроить свои категории или сбросить их к стандартным.

### В облаке (GitHub Pages)
Приложение доступно онлайн по адресу: `https://zemuro.github.io/fun-carousel/`

## 📝 Настройка
В приложении есть встроенная админ-панель (`admin.html`), где можно:
*   Добавлять и удалять категории занятий.
*   Наполнять категории (например, список игр, мультфильмов или творческих идей).
*   Восстанавливать стандартный набор данных.
